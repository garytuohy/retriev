Webhook Configuration
Add this endpoint to your Stripe dashboard

To receive real-time payment events, add this webhook URL to your Stripe dashboard under Developers → Webhooks.

https://retriev-api.garytuohy.workers.dev/api/webhook

Events to enable:

✓ payment_intent.payment_failed
✓ invoice.payment_failed
✓ customer.subscription.deleted
✓ invoice.paid

💡 Tip: In Stripe Dashboard, go to Developers → Webhooks → Add endpoint. Paste the URL above and select the events listed.
